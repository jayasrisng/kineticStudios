# Changelog

All notable changes to Kinetic Studios are documented here. The project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-06-27

### Added

- Clean Unity 6000.3.17f1 project foundation with a minimal desktop package set.
- Bootstrap and StudioWorkspace scenes with a polished UI Toolkit shell.
- Desktop orbit, pan, zoom, and presentation camera presets.
- Anchor, rod/string, and weight placement with selection and deletion.
- Editable pendulum length, mass, gravity, damping, and initial angle.
- Wood, metal, rope, and glass material studies.
- Play, pause, and reset simulation controls.
- Guided Demo V1 walkthrough for portfolio recording.
- Contributor, citation, issue, pull-request, and release documentation.
