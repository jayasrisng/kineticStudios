# Kinetic Studios v0.1.0 — First Studio Prototype

Kinetic Studios v0.1.0 is the first public portfolio release: a desktop Unity vertical slice for composing, tuning, and presenting a kinetic pendulum in a purpose-built virtual studio.

## Highlights

- Place anchor, rod/string, and weight components.
- Create and tune an assembled pendulum.
- Edit five physical properties and four material studies.
- Play, pause, reset, and inspect motion from three camera presets.
- Run a polished guided walkthrough with `F9` in Play Mode.

## Requirements

- Unity 6000.3.17f1
- Desktop target (Windows, macOS, or Linux editor)

Open `Assets/KineticStudios/Scenes/Bootstrap.unity` and press Play.

## Scope

This release is an interaction and presentation prototype. It does not include save/load, undo/redo, generalized mechanisms, XR, or the planned web companion.
